In this project you will extend a basic ray casting renderer by adding illumination.  

Liam O'Donnell - lko27@nau.edu  
Munendra Pratap Choudhary - mp2893@nau.edu  

Usage:
make  
./raycast width height input.scene output.ppm  

Example:
./raycast 500 500 test.scene output.ppm 

Known Issues:
